#  Carbon-Aware Machine Learning

**Predict and minimize CO₂ emissions for ML pipelines using energy-efficient models.**

---

##  Problem Statement
Machine learning workloads consume significant energy, generating CO₂ emissions and environmental impact.  
Researchers and developers need tools to **measure, predict, and optimize carbon footprints** of ML models.

---

##  Solution
This project provides a **carbon-aware ML framework**:
- Regression models predict CO₂ emissions for ML tasks
- Compare RandomForest and XGBoost models
- Identify features impacting carbon efficiency
- Enable energy-optimized ML decision-making

---

##  Key Features
- Multivariate regression for CO₂ emissions
- Supports RandomForest & XGBoost models
- Feature importance analysis
- Real-time dataset simulation (8000+ entries)
- Residual and predicted vs actual visualization
